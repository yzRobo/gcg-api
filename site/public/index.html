The Gundam Card Game API, in JSON.

gcg-api is a free, open dataset and REST API for Bandai's Gundam Card Game - the Gundam TCG - scraped weekly from the official card database, served from the edge, no key required.

what's in the data

Every printing
base cards and alt-arts, keyed by product_id; alt-arts share a card_number.
Structured effects
keyword abilities and timing markers parsed into arrays; filter with ?keyword=.
Official rulings
per-card FAQ entries (question + date + citation link back to the official page) via ?include=rulings.
Bulk files
the entire dataset as NDJSON/JSON on GitHub Releases. Unmetered; the files are the source of truth.
OpenAPI
machine-readable spec at /openapi.json; import it into any client.
Self-serve keys
optional; raises the rate limit. Browser challenge, no account, no email.

questions

Is there a free Gundam TCG API?
This is one. No account, no email, no cost - keyless requests work immediately, and a free key only raises the rate limit.
Where can I download Gundam Card Game card data?
The whole dataset is a single NDJSON or JSON file on GitHub Releases. Bulk downloads are unmetered and never touch the API rate limits.
How current is the data?
A GitHub Action re-scrapes the official card database every Monday at 06:00 UTC and commits the result, so the repo history doubles as the changelog.
Does it include card images?
Metadata only. image_url points at Bandai's own servers - images are never copied or rehosted here.
Can I use it in a commercial project?
Yes. Code is MIT; the data compilation is ODbL - attribution required, and derivative databases you publish must share alike. Apps and sites built from the data stay yours.
Is this official?
No. It is an unofficial community project, not affiliated with or endorsed by Bandai.
